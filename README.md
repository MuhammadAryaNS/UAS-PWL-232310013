# SurkenBot - AIGuide
SurkenBot adalah asisten virtual untuk kawasan kuliner Suryakencana, Bogor.
## Cara Menjalankan Proyek :
Untuk menjalankan aplikasi ini, ikuti langkah-langkah berikut :
1. **Clone repository ini**.
2. **Instal Dependencies :**
   Buka terminal di dalam folder proyek, lalu jalankan perintah : npm install
3. **Konfigurasi Environment :**
   Buat file baru bernama .env di direktori utama, lalu tambahkan API Key dengan format : GEMINI_API_KEY=(Salin API Key Disini)
4. **Jalankan Aplikasi :**
   Jalankan server dengan perintah : npm start atau node server.js
5. **Buka browser dan Akses :**
   http://localhost:3000

Catatan Teknis :
- Folder node_modules tidak disertakan untuk menjaga ukuran repository tetap ringan dan menyesuaikan package secara otomatis.
- File .env tidak diunggah demi keamanan API Key Pribadi.
- **Jalankan :** node test-api.js untuk melihat model apa saja yang ada di API Key Anda.
